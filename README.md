### WebRTC Tutorial - Building a P2P Video Chat App
Medium Article: https://medium.com/p/2f53dc1d02f7

This repository is to accompany my second in the series Medium article about WebRTC and how to setup your own signalling server using WS as well as leveraging the available browser APIs in React to quickly build a P2P Video Chat app.
* **./signalling-server** - Node/Socket.io based signalling server. A very simple example on how to handle the SDP and ICE candidate exchange
* **./react-video-client** - A React App for the P2P Video chat setup - Covers the basics of how to use the browser WebRTC APIs
